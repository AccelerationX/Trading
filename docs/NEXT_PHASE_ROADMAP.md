# Next Phase Roadmap

## Direction

The system is now in a usable baseline state.

The next major direction is not more technical modules. It is:

- stronger information interpretation
- stronger A-share sentiment framing
- clearer theme-to-stock mapping
- lower reliance on pure technical ranking

## Work order

### Step 1

Implement the sentiment layer:

- sentiment cycle
- leader stability
- event-driven bias

### Step 2

Implement information-first candidate ranking:

- information edge score
- lower priority for module-only ideas
- stronger priority for official events and theme beneficiaries

### Step 3

Build theme-board output:

- main themes
- probable leaders
- secondary beneficiaries
- low-quality followers to avoid

### Step 4

Expand information sources:

- better official policy feeds
- faster official news feeds
- richer announcement interpretation

### Step 5

Evaluate recommendation families separately:

- information-driven
- module-driven
- mixed resonance

The goal is to know which path actually produces useful decisions in live usage.
